# Propósito
O propósito com a criação desse site é poder estudar de forma didática e prática a linguagem ```JavaScript``` e poder também aprender mais sobre a construção de sites com ```HTML5``` e ```CSS3```.

# Jogos disponíveis
A princípio a ideia é de que o site disponibilize 3 jogos, sendo eles: xadrez, damas e jogo da velha. Três jogos simples, que todo mundo jogou ao menos uma vez na vida, que são divertidos e até mesmo desafiadores.
